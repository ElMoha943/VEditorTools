# VEditorTools
A collection of editor tools to streamline unity development, specifically for VRChat worlds.
You can find all the tools on the top bar under `ValenVRC/Tools/`

## Texture Importer
Using the texture importer you can define multiple rules that are used to mass change the import settings of your textures.
